# react_pages

First React page written directly in an HTML file using CDN links — no build tool or npm needed.

## Files

| File | Description |
|---|---|
| `firstReactPage.html` | React component loaded via CDN, renders a live clock |

## What I Learned

- Loading React, ReactDOM, and Babel directly from CDN in an HTML file
- Writing JSX inside a `<script type="text/babel">` tag
- Using `ReactDOM.render()` to mount a component to the DOM
- Using `setInterval()` to update the UI every second
