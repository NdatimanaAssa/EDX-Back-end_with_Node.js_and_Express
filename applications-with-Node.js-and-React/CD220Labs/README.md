# CD220Labs

Labs completed as part of the **Back-end Application Development with Node.js and Express** course on EDX.

## Folder Structure

```
CD220Labs/
├── http_server/          ← Raw HTTP server with Node.js built-in module
├── async_callback/       ← Async patterns: callbacks, promises, async/await
├── expressjs/            ← Express.js server, routing, middleware, authentication
├── react_pages/          ← First React page using CDN (no build tool)
├── react-apps/           ← React app scaffolded with Create React App
├── react_class_component/← React class components vs functional components
├── props_and_states/     ← React props and state management
└── todoapp/              ← Todo app built with React
```

## Labs Summary

| Lab | Key Concept |
|---|---|
| `http_server` | `http.createServer()`, `server.listen()`, custom modules |
| `async_callback` | Callbacks, Promises, Async/Await, Axios, JSON parsing |
| `expressjs` | Express routing, middleware, JWT, sessions, static files |
| `react_pages` | React via CDN, `ReactDOM.render()`, Babel in browser |
| `react-apps` | Create React App, project structure, JSX |
| `react_class_component` | Class vs functional components, `render()` |
| `props_and_states` | Props, state, UI re-rendering |
| `todoapp` | Full todo app, list state management |
